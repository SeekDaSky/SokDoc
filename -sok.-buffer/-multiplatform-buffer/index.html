MultiplatformBuffer

abstract class MultiplatformBuffer (source)

Platform and version requirements: Common

A MultiplatformBuffer is the primitive type of Sok, you will use it to receive, send and manipulate data

MultiplatformBuffer is a class each platform must implement to abstract native buffer types (ByteBuffer on JVM, Buffer on Node.js and ByteArray on Native). In order to have the most consistent behaviour and avoid platform-specific exception leak the below class implement all the behavioural code (cursor management, exception throwing) and adds abstract protected methods for platform specific code.

THIS CLASS IS NOT THREAD SAFE but you should obviously not be working with the same buffer on two different threads at the same time
