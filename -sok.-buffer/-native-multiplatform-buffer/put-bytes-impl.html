putBytesImpl

fun putBytesImpl(array: ByteArray, index: Int?): Unit (source)

Platform and version requirements: Native

Put the given byte array inside the buffer starting at the buffer cursor position. If the index parameter is provided, the cursor will be ignored and not modified

Parameters

array - data to put in the buffer

index - index of the first byte, buffer.cursor is used if the index is null
