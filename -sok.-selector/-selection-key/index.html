SelectionKey

class SelectionKey (source)

Platform and version requirements: Native

A SelectionKey is generated when a socket is registered to a selector. It is used to perform the suspention needed by the Client/Server socket. The Selector class use it to get the Continuation and resume it while updating the key interest accordingly
